使用帮助 & 指南

快速开始

在主页输入框中输入以下任意一种目标格式，然后点击"开始测试"：

  • Web 应用: https://example.com
  • GitHub 仓库: https://github.com/giaoimgiao/repo
  • 本地路径: C:\path\to\your\project

Agent 模式

ULRATTACK 提供三种 AI Agent 模式：

  • 渗透测试: 全面的安全漏洞扫描，生成 CVSS 评分报告。适合日常安全检查和合规审计。
  • 攻击测试: 基于发现的漏洞进行实际利用，验证攻击可行性。适合深度安全评估和红队测试。
  • 网络爬虫: 智能数据爬取，根据你的需求自动提取网页数据。需要先描述爬取需求。

扫描深度 (渗透/攻击模式)

  • Quick Scan: 快速检查常见漏洞，速度最快。
  • Standard Scan: 默认推荐，平衡深度与时间。
  • Deep Scan: 深度分析，包含复杂的逻辑链漏洞测试。

网络爬虫使用说明

使用爬虫模式时：

  1. 选择"网络爬虫"模式
  2. 在"爬取需求"文本框中描述你想要的数据，例如：
    • "爬取商品名称、价格和评分"
    • "获取所有新闻标题和发布时间"
    • "提取用户评论内容"
  3. 输入目标 URL 并开始爬取

交互模式

任务开始后，你可以通过底部的聊天输入框与 AI Agent 直接对话：

  • 询问进度或发现的细节
  • 提供特定的指令，例如"专注于 SQL 注入测试"
  • 要求 AI 针对某个发现编写 PoC

配置说明

点击侧边栏的 Settings 可以配置你自己的 LLM (大模型) API 密钥。支持 OpenAI, Anthropic, Gemini 等多种提供商。
